# Heath Renfroe

Finance major, econ minor at Oklahoma State (Dec 2026) | I build data pipelines,
forecasting models, and AI-driven automations, then publish the math and the
backtests behind them.

## Projects

**[oklahoma-desk](https://github.com/heathrenfroe-sys/oklahoma-desk)** — a
one-person regional research desk for Oklahoma: automated FRED/FDIC data
pipelines feeding a self-updating economic dashboard, ARIMA forecasts with
published backtests, and 50-state rankings.

**[quant-pi](https://github.com/heathrenfroe-sys/quant-pi)** — an autonomous
LLM paper-trading agent running on a Raspberry Pi 5: a Claude tool-use loop with
hard risk caps, an Alpaca broker layer, a curated Obsidian knowledge base it
reads as its "brain," and a 5-inch touchscreen dashboard.

**[blueprint-10k](https://github.com/heathrenfroe-sys/blueprint-10k)** — an MCP
server that gives Claude deterministic finance tools for SEC 10-K analysis:
EDGAR retrieval, ratio/FCF/VaR/options math, econometrics, and news sentiment —
driven by a multi-agent research pipeline.

**[make-automations](https://github.com/heathrenfroe-sys/make-automations)** —
no-code automation builds on Make.com, including a fully automated daily MLB
win-probability pipeline that posts model-generated predictions to Instagram.
